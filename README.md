# Portfolio Investment System

## 1. Summary
**Portfolio Investment System** is a business domain core designed to manage investment portfolios. It executes asset purchases, maintains a transaction log, and validates critical business rules using pure Java (Java 21 LTS). This project was developed as part of a Java course focused on Test-Driven Development (TDD)

## 2. Architecture
This project implements a **Pure Entity Core** following Clean Architecture (Ports & Adapters) principles. The core business domain is completely isolated from external frameworks, user interfaces, and databases.

## 3. Tech Stack & Standards
* **Language:** Java 21 LTS
* **Testing & Coverage:** JUnit 5, Mockito Core, and JaCoCo Maven Plugin.
* **Standards:** Strict AAA (Arrange, Act, Assert) unit testing pattern, and 100% mathematical coverage on the domain core.

## 4. Execution Commands
To run the automated test suite and check coverage from the terminal, execute the following commands:

* **To run the test suite and view the console summary:**
  `mvn clean test`

* **To generate the detailed HTML coverage report:**
  `mvn jacoco:report`

## 5. Evidence Path
After executing the report command, you can open and review the interactive report generated by JaCoCo at the following path:
`target/site/jacoco/index.html`
